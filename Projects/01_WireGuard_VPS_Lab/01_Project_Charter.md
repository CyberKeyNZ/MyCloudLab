# CloudLab Project 01 – Personal WireGuard VPS

## Status
Planned – Near Future

## Objective
Build and document a personal WireGuard VPN hosted on a VPS (Sydney/Auckland) to:

- Understand VPN architecture deeply
- Learn Linux networking and firewall rules
- Implement key-based authentication
- Practice NAT and routing
- Document full deployment as an operator-grade runbook

## Learning Goals
- IP forwarding & NAT
- WireGuard key exchange
- VPS provisioning
- Firewall hardening (UFW / iptables)
- Performance tuning (MTU, latency testing)

## Why
Learning-focused project aligned with CloudLab and Zero Trust mindset.
