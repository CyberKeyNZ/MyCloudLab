🛡️ SY0-701 Objective 1.2: Security Control Categories & Types
1. Control Categories (The "Who/What")
* Managerial (Administrative): Focused on design and policy. (e.g., Risk assessments, security awareness training).

* *perational: Focused on day-to-day human activities. (e.g., Log reviews, physical security guards).

* Technical (Logical): Security implemented through hardware or software. (e.g., Firewalls, Antivirus, IPS).

* Physical: Tangible items that limit access to a site. (e.g., Fences, locks, mantraps).

2. Control Types (The "Function")
* Preventive: Stops a security event before it happens. (e.g., Firewall, Door lock).

* Detective: Identifies and records a security event as it happens or after. (e.g., Logs, CCTV).

* Corrective: Reverses the impact of an event. (e.g., Restoring from backups, patching).

* Deterrent: Discourages a potential attacker. (e.g., "Warning" signs, lighting).

* Compensating: A "Plan B" when the primary control isn't feasible. (e.g., Using a separate vault if the main safe is broken).
