🛡️ Objective 1.1: General Security Concepts
1. The CIA Triad
* Confidentiality: Encryption, Access Control.

* Integrity: Hashing, Digital Signatures.

* Availability: Redundancy, Backups, Patching.

2. Security Control Categories
* Technical: Firewalls, Antivirus.

* Managerial: Policies, Training.

* Operational: Physical guards, log reviews.

* Physical: Locks, Fences.

3. Control Types
* Preventive: Stops the event (Firewall).

* Detective: Identifies the event (CCTV/Logs).

* Corrective: Fixes the impact (Backups).

* Deterrent: Discourages the attacker (Signs).
